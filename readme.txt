PURPOSE
Get high rated movies based on the country specified

DESCRIPTION
This is a python program coded in python 3. This program goes to the website "https://agoodmovietowatch.com/". This website has a list of high rated movies that are available according to the country. This program would create a csv file with the name, description and Genre of the movie

DOCUMENTATION

Country - to change to your country, change the COUNTRY constant to 'tag-[country]-netflix' Eg france would be 'tag-france-netflix'
Alarm - This program when executed runs an alarm function that opens a video in Chrome. You can change this video by changing the link in the CONSTANT VIDEO
